## 一、需求背景

### 1.1 现状分析
当前小瑶刺绣 AI 问答系统仅支持文字交互，用户需要阅读文字才能获取信息。考虑到：
- 部分用户喜欢"听"内容，尤其在通勤、做家务等场景
- 瑶绣知识具有浓厚的民族文化特色，语音朗读更能传递情感
- 竞品（如豆包、文心一言）均已支持**实时流式语音朗读**

### 1.2 竞品分析：豆包朗读特点
- **实时性**：文字流式生成的同时，语音同步输出
- **低延迟**：用户几乎感受不到文字和语音的间隔
- **高亮同步**：朗读到的文字实时高亮显示
- **自然流畅**：句子之间衔接平滑，无断裂感

### 1.3 目标用户
- 通勤族：上下班路上听瑶绣知识
- 视觉疲劳用户：长时间阅读后希望听音频休息眼睛
- 学习偏好型用户：喜欢通过听来学习记忆
- 老年用户：视力不好，更依赖听觉

### 1.4 价值主张
- **情感连接**：小瑶的声音让人设更立体，增强亲切感
- **场景扩展**：覆盖更多使用场景（开车、运动、睡前等）
- **体验升级**：实时流式朗读，媲美原生 AI 助手体验

## 三、技术方案详解

### 3.1 方案对比

| 方案 | 实时性 | 延迟 | 音质 | 成本 | 推荐度 |
|-----|-------|-----|------|-----|-------|
| **整段合成后再播放** | ❌ 需等文字生成完 | 3-10s | 高 | 低 | ⭐⭐ |
| **流式分段合成** | ✅ 边生成边读 | <1s | 高 | 中 | ⭐⭐⭐⭐⭐ |
| **Web Speech API** | ✅ 实时 | 0s | 低 | 免费 | ⭐⭐⭐ |

### 3.2 推荐方案：流式分段合成

#### 3.2.1 原理
```
AI 文字流式生成            语音合成              音频播放
     │                        │                      │
     ├── "(眼睛一亮)哎呀~" ───┼──→ 合成音频1 ──────┼──→ 播放
     ├── "你问到这个回纹呀" ──┼──→ 合成音频2 ──────┼──→ 播放

     │                        │                      │
```

### 3.3 推荐技术栈：Minimax 海螺语音

**理由**：
1. **国内首家**支持流式语音合成的 AI 服务商
2. **延迟极低**：首包延迟 < 500ms
3. **音色自然**：基于大模型，情感表达丰富
4. **支持克隆**：可克隆"小瑶"专属音色（后续可定制）
5. **价格适中**：15元/万字符

### 3.4 备选方案：Web Speech API + 分段优化

如果暂时不想接入第三方服务，可以用浏览器 API 优化实现：

```javascript
// 方案：按句子分段朗读
class StreamingTTS {
    constructor() {
        this.synth = window.speechSynthesis;
        this.queue = [];      // 待朗读文本队列
        this.isPlaying = false;
    }
    
    // 实时接收流式文字
    appendText(text) {
        // 按标点切分句子
        const sentences = text.split(/([。！？.!?])/);
        
        for (let sent of sentences) {
            sent = sent.trim();
            if (!sent || /^[。！？.!?]$/.test(sent)) continue;
            
            // 清洗文本
            sent = this.cleanText(sent);
            
            if (sent) {
                this.queue.push(sent);
            }
        }
        
        if (!this.isPlaying) {
            this.playNext();
        }
    }
    
    // 朗读下一段
    playNext() {
        if (this.queue.length === 0) {
            this.isPlaying = false;
            return;
        }
        
        this.isPlaying = true;
        const text = this.queue.shift();
        
        const utter = new SpeechSynthesisUtterance(text);
        utter.lang = 'zh-CN';
        utter.rate = 1.1;  // 稍快一点，接近流式感
        
        utter.onend = () => {
            this.playNext();
        };
        
        this.synth.speak(utter);
    }
    
    // 文本清洗
    cleanText(text) {
        return text
            .replace(/\*\*/g, '')
            .replace(/\([^)]*\)/g, '')
            .replace(/[🧵✨🌺💪😌]/g, '')
            .trim();
    }
}
```

```javascript
    // 1. 按句子切分（优先）
    const sentences = text.split(/([。！？\.\?\!])/);
```

#### 4.2.3 音频预缓冲策略
```javascript
class AudioBuffer {
    constructor() {
        this.chunks = [];        // 音频片段队列
        this.currentIndex = 0;   // 当前播放索引
        this.preloadCount = 2;   // 预加载数量（提前合成2句）
    }
    
    // 添加音频片段
    addChunk(text, audioData) {
        this.chunks.push({ text, audioData, duration: 0 });
        this.preloadNext();
    }
    
    // 预加载下一段
    async preloadNext() {
        const nextIndex = this.currentIndex + this.preloadCount;
        if (nextIndex < this.chunks.length && !this.chunks[nextIndex].audioData) {
            // 提前合成下下句
            await this.synthesize(this.chunks[nextIndex].text);
        }
    }
    
    // 获取下一段播放
    getNext() {
        if (this.currentIndex >= this.chunks.length) return null;
        return this.chunks[this.currentIndex++];
    }
}
```

```javascript
// 核心类设计
class StreamingReader {
    constructor() {
        this.synth = window.speechSynthesis;
        this.textBuffer = new TextBuffer();  // 文本缓冲
        this.audioQueue = new AudioQueue();  // 音频队列
        this.isPlaying = false;
        this.highlightElement = null;
    }
    
    // AI 文字流式生成时调用
    onTextStream(text) {
        // 清洗文本
        const cleanText = this.clean(text);
        // 按句切分
        const sentences = this.split(cleanText);
        // 加入队列
        sentences.forEach(s => this.textBuffer.push(s));
        // 开始朗读
        if (!this.isPlaying) this.start();
    }
    
    // 开始朗读
    async start() {
        this.isPlaying = true;
        while (this.textBuffer.hasNext()) {
            const sentence = this.textBuffer.next();
            // 合成并播放
            await this.speak(sentence);
            // 高亮显示
            this.highlight(sentence);
        }
        this.isPlaying = false;
    }
}
```

## 六、技术难点与解决方案

### 6.1 难点一：文字和语音同步
**问题**：AI 文字生成速度 vs 语音合成速度不一致

**方案**：
```javascript
// 速度同步策略
class SyncController {
    constructor() {
        this.textSpeed = 0;    // 文字生成速度（字/秒）
        this.ttsSpeed = 0;     // 语音合成速度（字/秒）
        this.bufferSize = 0;   // 缓冲队列长度
    }
    
    // 动态调节
    adjust() {
        if (this.textSpeed > this.ttsSpeed * 1.5) {
            // 文字生成太快，加速 TTS 或增加预合成
            this.preloadMore();
        } else if (this.textSpeed < this.ttsSpeed * 0.5) {
            // 文字生成慢，降低 TTS 语速或等待
            this.slowDownTTS();
        }
    }
}
```

### 6.2 难点二：句子衔接自然
**问题**：分段朗读时，句子之间有明显停顿

**方案**：
1. **预缓冲**：提前合成下一句，无缝衔接
2. **音频淡入淡出**：前一句淡出，后一句淡入
3. **智能切分**：尽量在标点处切分，避免语义断裂

### 6.3 难点三：文本清洗与位置映射
**问题**：清洗后的文本长度变化，导致高亮位置对不上

**方案**：
```javascript
// 位置映射表
class PositionMap {
    constructor() {
        this.mappings = [];  // [{raw: 0, clean: 0}, ...]
    }
    
    // 建立映射
    build(rawText) {
        let cleanIndex = 0;
        for (let i = 0; i < rawText.length; i++) {
            const char = rawText[i];
            // 跳过需要清洗的字符
            if (this.shouldSkip(char)) {
                this.mappings.push({ raw: i, clean: -1 });
            } else {
                this.mappings.push({ raw: i, clean: cleanIndex });
                cleanIndex++;
            }
        }
    }
    
    // 清洗文本位置 → 原始文本位置
    cleanToRaw(cleanPos) {
        const mapping = this.mappings.find(m => m.clean === cleanPos);
        return mapping ? mapping.raw : -1;
    }
}
```

## 七、API 接口设计

### 7.1 Minimax 流式 TTS 接入

```javascript
// MinimaxTTS.js
class MinimaxTTS {
    constructor(apiKey) {
        this.apiKey = apiKey;
        this.ws = null;
        this.audioQueue = [];
    }
    
    connect() {
        this.ws = new WebSocket('wss://api.minimax.chat/v1/t2a_v2?GroupId=xxx');
        
        this.ws.onopen = () => {
            console.log('Minimax TTS 连接成功');
        };
        
        this.ws.onmessage = (event) => {
            const data = JSON.parse(event.data);
            
            // 收到音频数据
            if (data.data && data.data.audio) {
                const audioHex = data.data.audio;
                const audioBuffer = this.hexToBuffer(audioHex);
                this.onAudioChunk(audioBuffer);
            }
            
            // 合成完成
            if (data.data && data.data.status === 'finished') {
                this.onComplete();
            }
        };
    }
    
    // 发送文本合成
    synthesize(text, options = {}) {
        const payload = {
            model: 'speech-01',
            text: text,
            stream: true,
            voice_setting: {
                voice_id: options.voiceId || 'female-yujie',
                speed: options.speed || 1.0,
                vol: options.vol || 1.0,
                pitch: options.pitch || 0
            },
            audio_setting: {
                sample_rate: 32000,
                bitrate: 128000,
                format: 'mp3',
                channel: 1
            }
        };
        
        this.ws.send(JSON.stringify(payload));
    }
    
    // 十六进制转 ArrayBuffer
    hexToBuffer(hex) {
        const bytes = new Uint8Array(hex.length / 2);
        for (let i = 0; i < hex.length; i += 2) {
            bytes[i / 2] = parseInt(hex.substr(i, 2), 16);
        }
        return bytes.buffer;
    }
}
```

### 10.3 文本清洗工具
```javascript
const TextCleaner = {
    // 完整清洗流程
    clean(text) {
        return text
            .replace(/\*\*/g, '')                    // Markdown 加粗
            .replace(/\([^)]*\)/g, '')               // 括号动作
            .replace(/[🧵✨🌺💪😌]/gu, '')            // Emoji
            .replace(/[\n\r]+/g, ' ')                // 换行
            .replace(/\s+/g, ' ')                   // 多余空格
            .trim();
    },
    
    // 保留映射关系（用于高亮）
    cleanWithMap(text) {
        const map = [];
        let cleanText = '';
        let cleanIndex = 0;
        
        for (let i = 0; i < text.length; i++) {
            const char = text[i];
            
            // 跳过清洗字符
            if (/[\*\(\)🧵✨🌺💪😌]/.test(char)) {
                map.push({ raw: i, clean: -1 });
                continue;
            }
            
            // 换行转空格
            if (/[\n\r]/.test(char)) {
                if (cleanText[cleanText.length - 1] !== ' ') {
                    cleanText += ' ';
                    map.push({ raw: i, clean: cleanIndex });
                    cleanIndex++;
                } else {
                    map.push({ raw: i, clean: -1 });
                }
                continue;
            }
            
            cleanText += char;
            map.push({ raw: i, clean: cleanIndex });
            cleanIndex++;
        }
        
        return { text: cleanText.trim(), map };
    }
};
```
