# 瑶绣·智问 v4.0 心理学驱动 redesign PRD

## 核心转变：从"工具"到"心理体验"

> 人不缺获取信息的渠道，人缺的是**被满足的心理体验**。

---

## 一、目标人群画像（三类最容易传播的核心用户）

### 🎯 第一类：「文化猎奇探索者」（占比40%）

**画像特征：**
- 18-30岁，一二线城市年轻白领/学生
- 喜欢新鲜事物，追求"小众高级感"
- 小红书重度用户，热衷分享"冷门知识"
- 对民族文化有猎奇心理，但不会深度学习

**心理动机：**
| 需求 | 心理机制 | 产品对应 |
|------|----------|----------|
| 猎奇感 | 寻求新奇刺激，满足探索欲 | 随机纹样盲盒、每日冷知识 |
| 优越感 | "我知道你不知道的" | 稀有纹样解锁、探索成就 |
| 社交货币 | 有内容可发朋友圈/小红书 | 一键生成精美分享卡片 |

**激活场景：**
- 午休刷手机时："今天瑶绣给我科普了啥？"
- 朋友聚会时："你知道吗，瑶族八角花其实是..."

---

### 🎯 第二类：「治愈系情绪消费者」（占比35%）

**画像特征：**
- 22-35岁，高压职场人群
- 需要"心理避风港"，喜欢ASMR、冥想类内容
- 对传统手工艺有"治愈滤镜"
- 愿意为"情绪价值"付费

**心理动机：**
| 需求 | 心理机制 | 产品对应 |
|------|----------|----------|
| 减压 | 看刺绣过程释放压力 | 沉浸式针法动画、ASMR音效 |
| 陪伴 | 孤独时需要"人"陪伴 | 傲娇小瑶人设、每日问候 |
| 掌控感 | 完成一件事的满足感 | 虚拟绣品完成、进度可视化 |

**激活场景：**
- 深夜加班后："看会儿小瑶绣花放松下"
- 周末独处时："今天完成一幅虚拟绣品"

---

### 🎯 第三类：「身份认同建构者」（占比25%）

**画像特征：**
- 25-40岁，设计师/文创从业者/文化爱好者
- 注重个人品味和审美表达
- 需要"文化深度"来建构自我形象
- 愿意为"专业身份"付费

**心理动机：**
| 需求 | 心理机制 | 产品对应 |
|------|----------|----------|
| 专业认同 | "我是懂这个的" | 深度知识解锁、专家认证 |
| 创造力 | 将知识转化为作品 | AI纹样生成、配色方案导出 |
| 影响力 | 被认可为领域达人 | 作品展示墙、被收藏统计 |

**激活场景：**
- 设计项目前："查一下瑶绣纹样做灵感参考"
- 社交场合："我最近研究瑶族刺绣，发现..."

## 二、功能重构：每个功能对应的心理学原理

### 【新增】🎲 纹样盲盒机

**功能描述：**
每日3次免费机会，点击"抽纹样"，随机解锁一个瑶绣纹样图鉴。

**心理学原理：可变奖励（斯金纳箱）**
- 固定奖励会腻，随机奖励最上瘾
- 人不是为了奖励而抽，是为了"期待奖励的感觉"

**上瘾机制设计：**
```
抽卡 → 动画 suspense → 揭晓 → 可能获得:
  ├─ 普通纹样 (60%) - 维持基础多巴胺
  ├─ 稀有纹样 (30%) - 小惊喜
  ├─ 史诗纹样 (9%) - 强烈满足
  └─ 传说纹样 (1%) - 炫耀级成就
```

**社交传播点：**
- 抽到稀有纹样时，自动弹出"炫耀一下"按钮
- 分享语："今天抽到了xx纹样，1%概率！"

**沉没成本设计：**
- 连续签到增加抽卡次数
- 收藏越多纹样，解锁隐藏图鉴

---

### 【新增】💝 情绪治愈室

**功能描述：**
一个独立的沉浸式空间，展示针法动画+白噪音+治愈文案。

**心理学原理：即时反馈 + 确定性安全感**
- 确定的治愈体验 = 大脑的安全感
- 低认知负荷 = 放松状态

**设计要素：**
| 元素 | 心理学作用 | 具体设计 |
|------|-----------|----------|
| 丝绣动画 | 视觉冥想，降低焦虑 | 慢速针法演示，8秒一针 |
| 呼吸引导 | 生理层面放松 | 跟随绣娘呼吸节奏 |
| 治愈文案 | 情感共鸣 | "今天辛苦了，看会儿绣花吧" |
| ASMR音效 | 听觉按摩 | 真实的刺绣声音采样 |

**每日问候机制：**
```
首次打开："阿姐/阿哥，今天想先绣花还是先聊天？"
深夜打开："这么晚还不睡？来，陪你绣会儿花放松一下~"
连续3天："你连续来了3天啦，送你一张稀有纹样！"
```

---

### 【新增】🏆 绣娘身份成长体系

**功能描述：**
用户通过互动获得"绣娘等级"，从"学徒"到"传承大师"。

**心理学原理：自我认同 + 沉没成本**
- 等级 = "我是谁"的身份定义
- 投入越多（时间、等级），越难放弃

**等级设计：**
```
Lv.1 好奇学徒   → 浏览10条知识
Lv.2 初级绣娘   → 收藏20条内容
Lv.3 纹样学徒   → 解锁10个纹样
Lv.4 配色助手   → 生成3套配色方案
Lv.5 小瑶师妹   → 连续7天签到
Lv.6 资深绣娘   → 分享10张卡片
Lv.7 纹样专家   → 被他人收藏50次
Lv.8 传承大师   → 解锁全部隐藏内容
```

**每级特权：**
- 专属称号展示
- 解锁新主题皮肤
- 增加每日抽卡次数
- 解锁高级AI功能

---

### 【新增】🎨 AI纹样共创实验室

**功能描述：**
用户输入关键词（如"春天+桃花"），AI生成融合瑶绣风格的原创纹样。

**心理学原理：掌控感 + 创造力满足**
- 我能改变结果 = 我有掌控权
- 创造内容 > 消费内容

**交互设计：**
```
1. 选择基底纹样（八角花/回纹/水波纹）
2. 输入灵感关键词（季节/情绪/元素）
3. 调整参数（复杂度/色彩倾向）
4. AI生成3个方案供选择
5. 一键优化/重新生成
6. 保存到个人作品库
```

**成瘾点：**
- "再生成一次可能更好看"
- 分享到社区获得点赞
- 作品被官方收录的荣誉感

---

### 【改造】📚 智能收藏 → 我的绣谱

**功能重命名理由：**
"收藏夹"是工具思维，"绣谱"是文化认同思维。

**新增心理机制：**
| 原设计 | 新设计 | 心理差异 |
|--------|--------|----------|
| 收藏内容 | 绣制纹样 | 从"保存"到"参与创作" |
| 标签分类 | 纹样谱系 | 从"整理"到"建立传承" |
| 导出数据 | 装订成册 | 从"备份"到"成就展示" |

**视觉改造：**
- 收藏夹UI变成"翻书"动效
- 每收藏一条，"绣谱"厚度+1
- 满100条触发"装订仪式"动画

---

### 【改造】↗️ 分享卡片 → 文化名片

**功能重命名理由：**
让用户分享的是"我的文化品味"，不是"一个链接"。

**新增模板：**
1. **探索者名片**："我已解锁23个瑶绣纹样，超过89%用户"
2. **治愈打卡**："今晚和小瑶一起绣花，第12天"
3. **作品展示**："我用AI设计的纹样《春日》"
4. **知识金句**："黑的是山，红的是心——瑶绣配色哲学"

**稀缺性设计：**
- 限定节日皮肤（春节/三月三/中秋）
- 限时传说纹样（错过等一年）
- 个人专属编号（第xxx位传承者）

---

### 【保留但强化】💬 小瑶对话

**人格优化：**
当前人设是"傲娇小绣娘"，需要强化"陪伴感"。

**新增互动机制：**
```javascript
// 记忆系统
- 记住用户名字："阿姐，你来了~"
- 记住上次话题："上次你说的那个问题，我后来又想了..."
- 记住用户偏好："知道你爱看纹样，今天给你准备了..."

// 情绪识别
- 深夜发送 → "这么晚还不睡？注意身体呀"
- 连续发问 → "你今天问题好多，是在做研究吗？"
- 长时间沉默 → "还在吗？我在这儿等你呢"
```

## 三、删减功能清单

### 需要删除的功能

| 功能 | 删除理由 | 替代方案 |
|------|----------|----------|
| 对话历史列表 | 增加认知负担，非核心用户不使用 | 自动保存最近3个对话 |
| API设置面板 | 破坏沉浸感，技术感太强 | 完全隐藏，自动配置 |
| 知识图谱（复杂版） | 学习成本高，普通用户看不懂 | 简化版"纹样谱系树" |
| 多模型切换 | 选择焦虑，增加决策成本 | 智能自动选择 |
| 字体大小调节 | 干扰项，破坏视觉统一 | 默认适配系统设置 |

### 需要弱化的功能

| 功能 | 弱化方式 |
|------|----------|
| 图片识别 | 折叠到"更多"菜单，不主动展示 |
| 代码块渲染 | 移除代码高亮，保持文化调性 |
| 主题切换 | 只保留2种（日/夜），减少选择 |

## 四、动画系统重构：情绪触发器

### 核心原则
每个动画都必须回答：**这个动画让用户产生什么情绪？**

### 情绪动画清单

| 触发场景 | 动画效果 | 情绪目标 | 技术实现 |
|----------|----------|----------|----------|
| 首次进入 | 八角花从中心绽放，丝线缠绕成logo | 仪式感、期待感 | CSS+SVG动画 |
| 收到回复 | 丝绣打字效果，每字带"绣"音效 | 被认真对待 | 逐字显示+音频 |
| 收藏内容 | 内容"飞"入绣谱，书页翻动 | 成就感 | FLIP动画 |
| 抽纹样 | 盒子抖动 → 光芒四射 → 揭晓 | 紧张期待 | 3阶段动画 |
| 升级时 | 丝带缠绕，烟花绽放，称号高亮 | 荣耀感 | 粒子+CSS |
| 连续签到 | 日历翻页，印章盖下 | 确定性满足 | 翻页动画 |
| 分享成功 | 卡片飞出，花瓣飘落 | 分享喜悦 | 贝塞尔曲线 |
| 深夜访问 | 界面自动调暗，小瑶点灯 | 被关怀感 | 暗色过渡 |

### 音效设计
```
- 点击：轻微的丝线声 "咝~"
- 发送：针穿过布料 "噗"
- 收到：绣品完成 "叮~"
- 收藏：书页合上 "啪"
- 升级：铜锣响起 "咚~"
- 抽卡：盒子摇晃 + 揭晓音效
```

## 六、变现设计（心理账户）

### 免费层（90%用户）
- 基础问答
- 每日1次抽卡
- 基础纹样
- 普通分享卡片

**心理策略：** 先让用户"拥有"，再让他们"害怕失去"

---

### 会员层（9%用户）：「绣娘秘传」

**定价：¥18/月 或 ¥128/年**

**特权设计（心理账户）：**
| 特权 | 心理价值 | 实际成本 |
|------|----------|----------|
| 每日抽卡次数+3 | "更多机会" | 无 |
| 解锁全部传说纹样 | "稀缺收藏" | 无 |
| 专属金色主题 | "身份标识" | 低 |
| AI纹样生成无限次 | "创造力" | API成本 |
| 治愈室全部音效 | "情绪价值" | 低 |
| 专属客服小瑶 | "被重视" | 低 |

**转化触发点：**
- 抽到好纹样时："会员可立即再抽3次！"
- 生成纹样时："非会员今日次数已用完"
- 连续7天使用时："送你7天会员体验"

---

### 单次付费层（1%用户）

**限定礼包：**
- 「春节限定纹样包」¥6.99
- 「非遗联名皮肤」¥12.99
- 「个人专属AI绣品」¥29.99

**心理策略：** 限时+限量+专属编号

## 七、数据指标（心理学导向）

### 核心指标
| 指标 | 目标值 | 心理意义 |
|------|--------|----------|
| 次日留存 | >50% | 首次体验成功 |
| 7日留存 | >30% | 习惯养成成功 |
| 30日留存 | >15% | 身份认同建立 |
| 分享率 | >10% | 社交传播激活 |
| 平均会话时长 | >3分钟 | 沉浸感达标 |
| 收藏转化率 | >40% | 价值认同建立 |

### 关键行为指标
| 行为 | 健康阈值 | 预警信号 |
|------|----------|----------|
| 每日抽卡参与率 | >60% | <40%说明奖励不够吸引 |
| 升级用户数占比 | >20% | <10%说明成长体系失效 |
| 收藏后再次打开率 | >50% | <30%说明收藏没有价值 |
| 分享后回流率 | >15% | <5%说明分享内容不够吸引人 |

## 八、技术实现建议

### 优先实现（MVP）
1. **纹样盲盒机** - 核心上瘾机制
2. **绣娘等级体系** - 身份认同
3. **分享卡片2.0** - 传播引擎
4. **情绪动画系统** - 体验升级

### 延后实现
1. AI纹样生成（成本高）
2. ASMR音效系统（资源大）
3. 社交功能（复杂度大）

### 需要的技术支持
- 简单的后端用户系统（等级、抽卡记录）
- 更复杂的动画库（GSAP）
- 音频资源管理和播放

---

## 九、一句话总结

> **让用户不是为了"查资料"而来，而是为了"看今天小瑶给我准备了什么惊喜"而来。**

这就是从工具到心理体验的跃迁。
