## 一、首屏弹窗系统

### 1.1 功能描述
用户首次进入/刷新页面时，展示精美的弹窗引导

### 1.2 弹窗内容
```
┌─────────────────────────────────────┐
│  ✨ 欢迎来到绣韵幻境 ✨              │
│                                     │
│     [今日运势占卜球动画]             │
│                                     │
│  今日可免费抽取 3 次纹样             │
│  每次对话额外获得 1 次机会           │
│                                     │
│  [立即前往抽奖]  [稍后再说]          │
└─────────────────────────────────────┘
```

### 1.3 动画效果
- 背景：淡入的黑色遮罩
- 弹窗：从中心缩放弹出，带弹性动画
- 运势球：缓慢旋转，内有流动光芒
- 按钮：悬停发光效果

### 1.4 触发逻辑
- 首次访问：自动弹出
- 每日首次刷新：自动弹出（本地记录日期）
- 用户关闭后当日不再弹出

## 二、等级进化动画系统

### 2.1 进化触发条件
用户等级提升时自动播放

### 2.2 进化动画流程

#### 阶段1：预示（0.5s）
- 屏幕轻微震动
- 背景渐变为金色
- 播放神秘音效（可选）

#### 阶段2：爆发（1.5s）
- 中心出现巨大emoji对应等级的图标
- 粒子爆炸效果（金色+等级对应颜色）
- 八角花纹样从中心向外扩散
- 文字"等级突破！"逐字显现

#### 阶段3：展示（2s）
- 新等级徽章旋转入场
- 新等级名称+emoji放大展示
- 解锁提示（如"解锁新纹样池"）
- 光芒收敛，恢复界面

### 2.3 各等级视觉效果

| 等级 | Emoji | 主色调 | 粒子效果 | 特殊效果 |
|------|-------|--------|----------|----------|
| 学徒 | 🌱 | 嫩绿 | 叶子飘落 | 无 |
| 绣娘 | 🧵 | 天蓝 | 丝线飞舞 | 无 |
| 巧手 | 🐱 | 翠绿 | 猫爪印记 | 无 |
| 能手 | 🧘 | 紫色 | 冥想光环 | 无 |
| 师傅 | 🎨 | 红色 | 画笔挥洒 | 边框金色描边 |
| 大师 | 🏆 | 金色 | 奖杯闪光 | 背景金色粒子 |
| 传人 | 👑 | 青色 | 皇冠坠落 | 全屏光晕 |
| 宗师 | ✨ | 白金色 | 星辰闪烁 | 彩虹边框+全屏闪耀 |

## 三、纹样卡牌系统扩展

### 3.1 卡牌数量扩展
- **总数**: 30张（原11张）

### 3.2 稀有度分级调整

| 稀有度 | 数量 | 概率 | 颜色主题 | 视觉特征 |
|--------|------|------|----------|----------|
| 普通 ★ | 12张 | 50% | 灰白色 | 简洁线条 |
| 稀有 ★★ | 8张 | 30% | 蓝色 | 微光效果 |
| 史诗 ★★★ | 6张 | 15% | 紫色 | 流光边框 |
| 传说 ★★★★ | 3张 | 4% | 金色 | 金边+闪烁 |
| 神话 ★★★★★ | 1张 | 1% | 白紫金 | 彩虹光晕+全屏特效 |

### 3.3 卡牌视觉样式

#### 普通 ★
- 边框：1px 灰色
- 背景：纯色
- 纹样：单色线条

#### 稀有 ★★
- 边框：1px 蓝色微光
- 背景：淡蓝渐变
- 纹样：双色填充

#### 史诗 ★★★
- 边框：2px 紫色流光动画
- 背景：深紫渐变+星星点点
- 纹样：多彩填充+细节纹理

#### 传说 ★★★★
- 边框：3px 金色闪烁
- 背景：黑金渐变+金色粒子
- 纹样：精细绘制+金边
- 特效：获得时金光爆发

#### 神话 ★★★★★
- 边框：4px 彩虹流光
- 背景：白金紫渐变+彩虹光晕
- 纹样：极致精细+动态效果
- 特效：
  - 获得时全屏彩虹光芒
  - 卡牌悬浮动画
  - 特殊音效（可选）
  - 自动分享提示

## 四、抽卡获得渠道扩展

### 4.1 获得渠道

| 渠道 | 奖励次数 | 限制 | 提示方式 |
|------|----------|------|----------|
| 每日基础 | 3次 | 每日重置 | 首屏弹窗 |
| 对话奖励 | 1次/对话 | 无上限 | Toast提示 |
| 分享奖励 | 2次/分享 | 每日最多3次分享 | 分享成功弹窗 |
| 连续签到 | +1次 | 每3天连续签到 | 签到提示 |

- 支持分享到微信、QQ、微博
- 分享内容包含用户收集进度
- 分享后跳转回应用并发放奖励

```css
:root {
    /* 稀有度颜色 */
    --rarity-common: #95a5a6;
    --rarity-rare: #3498db;
    --rarity-epic: #9b59b6;
    --rarity-legendary: #f1c40f;
    --rarity-mythic: linear-gradient(135deg, #fff 0%, #e1bfff 50%, #ffd700 100%);
    
    /* 特效颜色 */
    --effect-gold: #ffd700;
    --effect-rainbow: linear-gradient(90deg, #ff0000, #ff7f00, #ffff00, #00ff00, #0000ff, #4b0082, #9400d3);
}
```

### 6.2 动画时长规范
- 微动效：150-200ms
- 标准动效：300ms
- 入场动效：500ms
- 庆祝动效：1500-2000ms
- 进化动效：4000ms

### 6.3 粒子效果规范
- 普通：10-20粒子
- 稀有：20-30粒子
- 史诗：30-50粒子
- 传说：50-80粒子+光柱
- 神话：80-100粒子+全屏特效
