# 小瑶刺绣AI v2.1.0 需求文档

## 1. 消息功能优化

### 1.1 重新生成修正
**问题**：当前重新生成会删除用户消息并重新发送
**修正**：
- 保留用户消息不动
- 只清空并重新生成AI回复框
- 视觉上：用户气泡保持，AI气泡进入loading状态重新生成

### 1.2 用户消息编辑功能
**交互设计**：
- 用户点击自己发送的消息气泡
- 气泡下方显示操作栏：左「编辑笔图标」右「删除图标」
- 点击编辑：气泡变成输入框，可修改内容，回车或失焦保存
- 点击删除：删除该用户消息及对应的AI回复（消息对）

**UI细节**：
- 笔图标：蓝色/主题色，hover放大
- 删除图标：红色，hover放大
- 默认隐藏，hover消息时显示提示

## 2. 绣谱系统（皮肤/头像系统）

### 2.1 概念澄清
- **收藏夹**：保存喜欢的对话（原有功能，保留）
- **绣谱**：皮肤/头像/背景更换系统（新增）

### 2.2 绣谱内容
**纹样卡片**：通过盲盒抽取获得
- 背景纹样：更换聊天界面背景
- 头像框：更换小瑶或用户头像边框
- 主题色：更换界面主题色

**稀有度**：日用/节庆/秘传/祖灵（与盲盒稀有度对应）

### 2.3 入口位置
1. **成长进度面板**（好奇学徒板块）
   - 等级卡片下方添加「我的绣谱」入口
   - 点击进入绣谱展示页面

2. **盲盒系统**
   - 盲盒结果弹窗添加「查看绣谱」按钮
   - 纹样图鉴入口（展示所有可获得纹样）

### 2.4 绣谱界面
- 网格展示已获得纹样
- 点击纹样预览效果（背景/头像/主题）
- 应用按钮确认更换
