## 如何启用 v2.0.0 功能

### 步骤1: 在 index.html 中引入新文件

在 `index.html` 的 `<head>` 中添加样式：

```html
<!-- v2.0.0 样式 -->
<link rel="stylesheet" href="assets/v2-styles.css">
```

在 `index.html` 的 `</body>` 前添加脚本：

```html
<!-- v2.0.0 核心系统 -->
<script src="assets/v2-system.js"></script>
<script src="assets/v2-ui.js"></script>

<!-- v2.0.0 初始化 -->
<script>
  // 初始化 v2.0.0 功能
  document.addEventListener('DOMContentLoaded', () => {
    // 初始化等级徽章
    LevelBadge.init();
    
    // 初始化签到组件
    CheckinComponent.init();
    
    // 添加盲盒入口按钮到 header
    const headerActions = document.querySelector('header .flex.items-center.gap-1');
    if (headerActions) {
      headerActions.insertAdjacentHTML('beforeend', `
        <button onclick="BlindBoxModal.open()" class="blindbox-entry-btn" title="得纹样">
          <span>🎁</span>
          <span class="blindbox-count">${V2System.blindBox.getDailyDraws()}</span>
        </button>
      `);
    }
    
    // 添加收藏册入口
    const favBtn = document.getElementById('headerFavBtn');
    if (favBtn) {
      favBtn.onclick = () => PatternCollection.open();
    }
  });
</script>
```

## 功能说明

### 1. 盲盒系统（得纹样）

**入口**: 点击 header 上的 🎁 按钮

**规则**:
- 每日3次免费抽卡机会（随等级增加）
- 纹样分为4个等级：日用纹/节庆纹/秘传纹/祖灵纹
- 抽到重复纹样转化为经验值

**动画效果**:
- 盒子悬浮动画
- 点击后抖动→光芒→打开
- 结果卡片飞入

### 2. 等级体系（绣娘资历）

**显示**: header 上的等级徽章

**等级列表**:
| 等级 | 称号 | 解锁条件 |
|------|------|----------|
| 1 | 好奇学徒 | 初始 |
| 2 | 初级绣娘 | 收集10个纹样 |
| 3 | 纹样学徒 | 收集20个纹样 |
| 4 | 配色助手 | 分享5次 |
| 5 | 小瑶师妹 | 连续签到7天 |
| 6 | 资深绣娘 | 收集40个纹样 |
| 7 | 纹样专家 | 收集50个纹样 |
| 8 | 传承大师 | 获得1个祖灵纹 |

**特权**:
- 等级越高，每日抽卡次数越多
- 高等级增加稀有纹样概率
- 解锁专属主题

### 3. 签到系统

**显示**: 页面顶部的签到条

**奖励**:
- 每日签到：+1抽卡机会
- 连续3天：+3抽卡机会
- 连续7天：稀有纹样 guaranteed
- 连续30天：史诗纹样

### 4. 纹样收藏册（我的绣谱）

**入口**: 点击 header 上的收藏按钮 ⭐

**功能**:
- 查看所有已收集纹样
- 按稀有度筛选
- 查看纹样故事和寓意

## 数据存储

所有数据存储在 localStorage 中：

| Key | 说明 |
|-----|------|
| `yao-level` | 当前等级 |
| `yao-xp` | 经验值 |
| `yao-unlocked-patterns` | 已解锁纹样ID列表 |
| `yao-daily-draws` | 今日已使用抽卡次数 |
| `yao-last-draw-date` | 最后抽卡日期 |
| `yao-last-checkin` | 最后签到日期 |
| `yao-checkin-streak` | 连续签到天数 |
