## 🎨 设计系统

### 色彩体系（CSS 变量）

```css
/* 核心色彩 - 黑底红彩瑶族传统 */
--bg-primary: #0a0a0b;      /* 主背景 - 深黑 */
--bg-secondary: #141416;    /* 次级背景 - 卡片 */
--bg-tertiary: #1f1f23;     /* 三级背景 - 输入框 */
--border-color: #2a2a2e;    /* 边框色 */
--accent: #c41e3a;          /* 主题红 - 按钮、强调 */
--accent-light: #e63950;    /* 亮红 - hover */
--gold: #d4a574;            /* 金色 - 装饰、点缀 */
--gold-light: #e8c9a0;      /* 浅金 */
--text-primary: #f5f5f5;    /* 主文字 - 白 */
--text-secondary: #a1a1aa;  /* 次级文字 - 灰 */
--text-tertiary: #71717a;   /* 三级文字 - 浅灰 */
```

### 主题变体

```
[data-theme="red"]    - 黑底红彩（默认）
[data-theme="blue"]   - 青花蓝调
[data-theme="light"]  - 素雅白底
[data-theme="auto"]   - 跟随系统
```

### 字体规范

```css
font-family: 'Noto Sans SC', sans-serif;  /* 正文 */
font-family: 'Noto Serif SC', serif;      /* 标题、瑶字 */
```

## 🧩 组件规范

### 玻璃态卡片

```html
<div class="glass rounded-2xl border p-4">
  <!-- 内容 -->
</div>
```

```css
.glass {
  background: rgba(255, 255, 255, 0.03);
  backdrop-filter: blur(20px);
  border-color: var(--border-color);
}
```

### 消息气泡

```html
<!-- 用户消息 -->
<div class="message-bubble ml-auto">
  <div class="rounded-2xl rounded-tr-sm px-5 py-4 border" 
       style="background: var(--accent); opacity: 0.1; border-color: var(--accent);">
    <p style="color: var(--text-primary);">内容</p>
  </div>
</div>

<!-- AI 消息 -->
<div class="message-bubble">
  <div class="rounded-2xl rounded-tl-sm px-5 py-4 border" 
       style="background: var(--bg-secondary); border-color: var(--border-color);">
    <div class="chat-content" style="color: var(--text-primary);">内容</div>
  </div>
</div>
```

### 按钮样式

```html
<!-- 主按钮 -->
<button class="btn-glow rounded-xl px-4 py-2 font-medium"
        style="background: var(--accent); color: white;">
  按钮文字
</button>

<!-- 次级按钮 -->
<button class="rounded-xl px-4 py-2 font-medium border"
        style="background: var(--bg-secondary); border-color: var(--border-color); 
               color: var(--text-secondary);">
  按钮文字
</button>
```

### 输入框

```html
<div class="relative rounded-2xl border focus-within:ring-2" 
     style="background: var(--bg-secondary); border-color: var(--border-color);">
  <textarea class="w-full bg-transparent px-5 py-4 resize-none focus:outline-none"
            style="color: var(--text-primary);"
            placeholder="询问瑶绣知识..."></textarea>
  <button class="absolute right-3 bottom-3 p-2.5 rounded-xl"
          style="background: var(--accent); color: white;">
    <!-- 发送图标 -->
  </button>
</div>
```

## 🎯 交互规范

### 过渡动画

```css
/* 所有颜色变化统一过渡 */
*, *::before, *::after {
  transition: background-color 0.3s ease, 
              border-color 0.3s ease, 
              color 0.3s ease;
}

/* 面板滑入 */
.panel {
  transform: translateX(100%);
  transition: transform 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}
.panel.open {
  transform: translateX(0);
}

/* 悬停效果 */
.hover-lift {
  transition: transform 0.2s ease;
}
.hover-lift:hover {
  transform: translateY(-2px);
}
```

### 打字光标

```css
.cursor-blink::after {
  content: '▊';
  color: var(--accent);
  animation: blink 1s step-end infinite;
}
```

## 📱 响应式断点

```
mobile:  < 640px   (单列，全屏面板)
tablet:   640-768px (适配)
desktop:  > 768px   (侧滑面板，75%消息宽度)
```

### 响应式类名规范

```html
<!-- 字体 -->
<p class="text-sm sm:text-base">内容</p>

<!-- 间距 -->
<div class="p-4 sm:p-6">内容</div>

<!-- 网格 -->
<div class="grid grid-cols-1 sm:grid-cols-3 gap-2 sm:gap-3">

<div class="w-10 h-10 sm:w-12 sm:h-12">

<!-- 显示/隐藏 -->
<span class="hidden sm:inline">只在桌面显示</span>
```

## 🎭 文化元素

### 瑶族刺绣符号

```html
<!-- 八角花 -->
<span style="color: var(--gold);">❋</span>

<!-- 十字 -->
<span style="color: var(--accent);">✦</span>

<!-- 菱形 -->
<span style="color: var(--gold-light);">◈</span>
```

### 文案风格

- **标题**：使用"瑶绣"而非"瑶族刺绣"，简洁有力
- **按钮**：动词前置，如"立即收藏"、"生成分享"
- **提示**：温和友好，如"正在思考..."、"已为您收藏"
- **计量**：用"纹"代替"条"，如"今日共绣 12 纹"

## 🚀 快速开发提示词

### 场景 1：添加新面板

```markdown
请添加一个【xxx】面板，要求：
1. 从右侧滑入，宽度 400px（移动端全屏）
2. 使用 glass 玻璃态样式
3. 主题色彩适配所有主题
4. 包含关闭按钮和标题
5. 点击遮罩层可关闭
```

### 场景 2：添加新按钮

```markdown
请在【位置】添加一个【功能】按钮，要求：
1. 使用 btn-glow 光效类
2. 背景色 var(--accent)，文字白色
3. 圆角 rounded-xl
4. 包含【图标】在左侧
5. hover 时有抬起效果
```

### 场景 3：添加新消息类型

```markdown
请添加一种新的 AI 回复卡片，用于展示【内容类型】，要求：
1. 左侧显示瑶绣专家头像（红底白字"瑶"）
2. 右侧消息气泡使用 message-bubble 类
3. 背景 var(--bg-secondary)，边框 var(--border-color)
4. 圆角 rounded-2xl rounded-tl-sm
5. 内部包含【具体内容】
6. 底部有收藏和分享按钮
```

### 场景 4：响应式适配

```markdown
请将【组件】适配移动端，要求：
1. 桌面端：当前样式保持不变
2. 平板端（md）：适当缩小间距
3. 移动端（sm）：全宽显示，内边距 p-4 → p-3
4. 小字体 text-base → text-sm
5. 隐藏非核心元素
```
