## 情绪触发设计

### 1. 期待感（盲盒抽取）

**视觉元素：**
- 盒子持续轻微抖动 `animation: shake 0.5s infinite`
- 光芒从缝隙中透出
- 点击后盒子剧烈摇晃
- 光芒爆发效果
- 最终缓缓打开

**动效时间线：**
```
0.0s - 点击
0.2s - 盒子剧烈摇晃（200ms）
0.5s - 光芒开始聚集
1.0s - 光芒爆发
1.2s - 盒子打开
1.5s - 卡片飞出
```

### 2. 成就感（升级时刻）

**视觉元素：**
- 全屏暗化聚焦
- 粒子从中心爆发
- 等级数字翻转动画
- 称号带光晕展示
- 解锁奖励逐个飞入

**配色变化：**
- 升级前：普通界面
- 升级中：金色光芒笼罩
- 升级后：新等级主题色

### 3. 治愈感（情绪治愈室）

**视觉元素：**
- 整体亮度降低 20%
- 饱和度降低 15%
- 呼吸灯效果边框
- 缓慢流动的背景
- 温暖的金色调

### 4. 荣耀感（分享卡片）

**视觉元素：**
- 金色边框
- 瑶族传统纹样装饰
- 专属编号
- 二维码
- 个人成就数据

## 动效规范

### 缓动函数
```css
/* 默认交互 */
--ease-default: cubic-bezier(0.4, 0, 0.2, 1);

/* 弹性效果（奖励、升级） */
--ease-elastic: cubic-bezier(0.68, -0.55, 0.265, 1.55);

/* 爆发效果（抽卡、庆祝） */
--ease-burst: cubic-bezier(0.16, 1, 0.3, 1);

/* 缓入（消失、离开） */
--ease-out: cubic-bezier(0.0, 0, 0.2, 1);
```

### 时长规范
```css
/* 微交互 */
--duration-fast: 150ms;      /* 按钮点击、hover */

/* 标准交互 */
--duration-normal: 300ms;    /* 面板切换、页面跳转 */

/* 强调动画 */
--duration-slow: 500ms;      /* 重要反馈、通知 */

/* 复杂动画 */
--duration-complex: 1000ms;  /* 抽卡、升级 */
```

### 动画库使用
```javascript
// 简单动画 - CSS
.element {
  transition: all var(--duration-normal) var(--ease-default);
}

// 复杂动画 - GSAP
import gsap from 'gsap';

gsap.to(element, {
  scale: 1.5,
  rotation: 360,
  duration: 0.5,
  ease: 'elastic.out(1, 0.3)'
});
```

## 无障碍设计

```css
/* 减少动画偏好 */
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}

/* 高对比度模式 */
@media (prefers-contrast: high) {
  :root {
    --text-primary: #ffffff;
    --text-secondary: #cccccc;
    --border-color: #666666;
  }
}
```

## 性能规范

- 动画元素使用 `transform` 和 `opacity`（GPU加速）
- 粒子效果数量限制在50个以内
- 复杂动画使用 `will-change` 提示
- 避免同时运行多个复杂动画
- 使用 `requestAnimationFrame` 进行逐帧动画
