# 瑶绣 AI 问答 - 设置功能需求文档

## 1. 功能概述

在现有单 HTML 页面基础上，增加侧滑/弹窗式设置面板，提供个性化配置和实用工具。

## 2. 功能模块

### 2.1 主题设置
- **色彩主题**：
  - 黑底红彩（默认，经典瑶族传统）
  - 青花蓝调（蓝靛瑶族支系风格）
  - 素雅白底（护眼模式）
  - 跟随系统
- **实现方式**：CSS 变量切换，保存到 localStorage

### 2.2 字体设置
- **字体大小**：小/中/大/特大 四档（14px/16px/18px/20px）
- **字重调节**：正常/中等/加粗
- **应用范围**：聊天内容、输入框、系统提示

### 2.3 API 配置
- **API URL 输入框**：可编辑，默认显示当前配置
- **API Key 输入框**：密码隐藏显示，可选保存到本地（风险提示）
- **连接测试按钮**：点击测试 API 连通性
- **重置默认**：一键恢复原始配置

### 2.4 数据管理
- **导出对话**：下载为 Markdown / JSON 格式
- **导入对话**：从 JSON 恢复历史记录
- **清空历史**：带二次确认，清空 localStorage 中的对话

### 2.5 关于信息
- 应用名称和版本
- 瑶族刺绣文化简介
- 技术栈说明
- 链接：项目源码、DeepSeek、Cloudflare

## 3. 交互设计

### 3.1 入口
- 位置：Header 右侧，新增齿轮图标按钮
- 点击：右侧滑出设置面板（移动端全屏）

### 3.3 动画效果
- 面板滑入：从右向左，300ms ease-out
- 遮罩层：黑色 50% 透明度，点击关闭
- 切换主题：平滑过渡，500ms
- 保存提示：底部 Toast，2秒自动消失

## 4. 技术实现

### 4.1 数据存储
```javascript
localStorage 键值：
- yao-theme: 'red' | 'blue' | 'light' | 'auto'
- yao-font-size: '14' | '16' | '18' | '20'
- yao-font-weight: 'normal' | 'medium' | 'bold'
- yao-api-url: string
- yao-api-key: string (AES简单加密或明文)
- yao-chat-history: Array (对话记录)
```

### 4.2 主题系统
```css
/* 黑底红彩 (默认) */
[data-theme="red"] {
  --bg-primary: #0a0a0b;
  --bg-secondary: #141416;
  --accent: #c41e3a;
  --accent-light: #e63950;
  --gold: #d4a574;
  --text-primary: #e5e5e5;
  --text-secondary: #a3a3a3;
}

/* 青花蓝调 */
[data-theme="blue"] {
  --bg-primary: #0a0f1a;
  --bg-secondary: #111827;
  --accent: #3b82f6;
  --accent-light: #60a5fa;
  --gold: #93c5fd;
  --text-primary: #e5e5e5;
  --text-secondary: #9ca3af;

/* 素雅白底 */
[data-theme="light"] {
  --bg-primary: #fafaf9;
  --bg-secondary: #f5f5f4;
  --accent: #c41e3a;
  --accent-light: #e63950;
  --gold: #b45309;
  --text-primary: #1c1917;
  --text-secondary: #57534e;
}
```

### 4.3 代码结构
- `index.html` 中新增设置面板 HTML
- 新增 `Settings` 对象管理配置逻辑
- 初始化时读取 localStorage 应用配置
- 主题切换动态更新 CSS 变量

## 5. 边界情况

- **首次使用**：无 localStorage，使用默认配置
- **API 配置错误**：测试连接失败时显示红色提示
- **导入失败**：文件格式错误时提示重试
- **移动端**：设置面板改为底部滑出或全屏
- **主题切换闪烁**：使用 transition 避免突兀
