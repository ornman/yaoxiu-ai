# 知识图谱创意交互动画设计

## 核心概念："绣线入画"

点击节点时，仿佛用针线把知识点从图谱中"绣"到对话框里。

---

## 动画流程

### 1. 悬停阶段（Hover）
```
鼠标移入节点
    ↓
节点轻微放大 1.15 倍（弹性效果）
    ↓
圆环出现金色描边流动动画（像绣线缠绕）
    ↓
文字微微上浮
```

### 2. 点击瞬间（Click Down）
```
鼠标按下
    ↓
节点缩小到 0.9 倍（蓄力感）
    ↓
圆环变白色
    ↓
准备发射
```

### 3. 点击释放（Click Up）- 核心动画
```
白色波纹从节点中心扩散（3 层）
    ↓
节点被"拉向"右侧主界面（贝塞尔曲线轨迹）
    ↓
飞行过程中：
    - 节点拉长变形（椭圆 → 矩形）
    - 颜色从节点色渐变到消息气泡色
    - 大小逐渐匹配对话框
    ↓
其他节点：
    - 快速向四周散开（退避效果）
    - 透明度降低
    ↓
连线：
    - 像断线一样抖动
    - 逐渐淡化
```

### 4. 变形完成（Transform）
```
飞行节点到达主界面位置
    ↓
完美变形为用户消息气泡
    ↓
"啵"的一声轻微弹跳
    ↓
触发 AI 回复流程
```

### 5. 场景切换（Scene Transition）
```
知识图谱面板：
    - 整体向左滑出
    - 或缩小到背景作为装饰
    ↓
主界面：
    - 从右侧滑入
    - 或放大到前景
```

## 技术实现要点

### 飞行轨迹计算
```javascript
// 起点：节点在图谱中的位置
const startX = node.px;
const startY = node.py;

// 终点：主界面输入框位置
const endX = window.innerWidth - 100;
const endY = window.innerHeight - 150;

// 贝塞尔曲线控制点（制造弧线效果）
const controlX = (startX + endX) / 2 + 100;
const controlY = startY - 200;
```

### 变形动画关键帧
```css
@keyframes flyToChat {
    0% {
        transform: translate(0, 0) scale(1);
        border-radius: 50%;
        background: var(--accent);
    }
    30% {
        transform: translate(50px, -30px) scale(0.8, 1.2);
        border-radius: 40%;
    }
    60% {
        transform: translate(150px, -50px) scale(1.5, 0.8);
        border-radius: 20px;
        background: var(--accent);
    }
    100% {
        transform: translate(var(--endX), var(--endY)) scale(1);
        border-radius: 20px 20px 20px 4px;
        background: var(--accent-light);
    }
}
```

### 波纹效果
```css
@keyframes whiteRipple {
    0% {
        transform: scale(0);
        opacity: 1;
        stroke-width: 3;
    }
    100% {
        transform: scale(3);
        opacity: 0;
        stroke-width: 0;
    }
}
```

### 其他节点退避
```javascript
// 非点击节点向远离中心方向移动
otherNodes.forEach(node => {
    const angle = Math.atan2(node.y - clickedNode.y, node.x - clickedNode.x);
    const distance = 100;
    node.targetX = node.x + Math.cos(angle) * distance;
    node.targetY = node.y + Math.sin(angle) * distance;
    node.targetOpacity = 0.3;
});
```

## 视觉细节

### 飞行轨迹丝线
- 节点飞行时拖出金色丝线
- 丝线像绣线一样逐渐消散

### 残影效果
- 节点飞行时有 2-3 个半透明残影
- 形成动态模糊感

### 终点提示
- 主界面输入框提前高亮
- 显示"准备接收..."微光效果

---

## 失败回退

如果用户取消或动画出错：
1. 节点弹回原来位置（弹性动画）
2. 其他节点恢复
3. 显示"点击取消"提示
