# 瑶绣 AI 问答 v2.0 产品需求文档

## 一、用户痛点分析

### 1.1 AI 对话类通用痛点
| 痛点 | 具体表现 | 影响 |
|------|----------|------|
| 信息密度低 | 回复冗长，重点不突出 | 阅读疲劳，效率低 |
| 缺乏可视化 | 纯文字描述纹样/针法，难理解 | 学习门槛高 |
| 内容无法沉淀 | 好的回答看完就丢，无法保存 | 价值流失 |
| 社交属性弱 | 有趣的内容无法分享 | 传播受限 |
| 被动学习 | 只能提问，缺乏系统引导 | 学习路径零散 |

### 1.2 非遗/手工艺类特殊痛点
| 痛点 | 具体表现 | 解决方向 |
|------|----------|----------|
| 工艺难以理解 | 八角花结构复杂，文字说不清 | 可视化拆解 |
| 缺乏实操感 | 只看文字，不知从何下手 | 步骤动画演示 |
| 知识碎片化 | 问一答一，缺乏体系 | 知识图谱导航 |
| 难以转化创作 | 学了不会用，无法生成设计 | AI 辅助生成 |

## 二、核心功能规划

### 2.1 功能优先级
```
P0（核心）: 纹样可视化展示、智能收藏夹、分享卡片
P1（增强）: 知识图谱导航、针法步骤演示
P2（进阶）: AI 纹样生成器、学习路径规划
```

### 2.2 详细功能设计

#### 【P0-1】纹样可视化展示
**需求描述**：AI 回复中自动识别纹样关键词，插入可交互的 SVG 可视化组件

**功能点**：
- 八角花纹样：几何结构拆解，可旋转查看
- 十字挑花：网格展示，标记挑花点位
- 配色方案：展示黑底红彩的经典配色表
- 悬停显示：鼠标悬停显示部位名称和寓意

**技术方案**：
- 内嵌 SVG 组件库（八角花、回纹、水波纹等）
- 使用 CSS transform 实现旋转/缩放
- 响应式设计，移动端适配

#### 【P0-2】智能收藏夹
**需求描述**：用户可收藏有价值的问答，支持标签管理和快速检索

**功能点**：
- 收藏按钮：每条 AI 回复下方显示收藏 ⭐ 按钮
- 收藏面板：侧滑面板展示所有收藏
- 标签系统：自动标签（纹样/针法/配色/历史）+ 自定义标签
- 搜索过滤：按关键词、标签筛选
- 导出功能：导出收藏为 PDF/Markdown

**存储方案**：
- IndexedDB 存储（容量大，支持结构化查询）
- 导出 JSON 备份

#### 【P0-3】分享卡片生成
**需求描述**：将精彩问答生成精美图片，便于社交媒体分享

**功能点**：
- 分享按钮：一键生成卡片
- 多种模板：
  - 经典模式：黑底红彩，八角花边框
  - 清新模式：浅色底，简约设计
  - 金句模式：只展示核心金句
- 二维码：卡片带链接二维码，扫码直达
- 一键下载：PNG 格式，适配微信朋友圈/小红书

**技术方案**：
- html2canvas 生成图片
- Canvas 绘制二维码

---

#### 【P1-1】知识图谱导航
**需求描述**：以图谱形式展示瑶绣知识体系，支持点击探索

**功能点**：
- 可视化图谱：节点为知识点，边为关联
- 中心节点：瑶族刺绣
- 一级分支：纹样、针法、配色、服饰、历史
- 二级展开：点击展开详细子项
- 快捷提问：点击节点直接提问

**交互设计**：
- 力导向图布局
- 支持拖拽、缩放
- 高亮关联路径

#### 【P1-2】针法步骤演示
**需求描述**：用 CSS 动画演示十字挑花等针法的具体步骤

**功能点**：
- 步骤分解：起针 → 挑花 → 收针
- 动画播放：自动播放/手动控制
- 网格标注：显示正反面视角
- 速度控制：慢速/正常/快速

### 4.2 数据存储结构
```javascript
// 收藏数据结构

  id: "uuid",
  question: "问题文本",
  answer: "回答文本（处理后）",
  tags: ["纹样", "八角花"],
  timestamp: 1712345678,
  // 分享卡片配置
  cardStyle: "classic" | "clean" | "quote"
}

// 知识图谱数据
{
  nodes: [
    { id: "embroidery", label: "瑶族刺绣", category: "root" },
    { id: "pattern", label: "纹样", category: "branch" },
    { id: "octagon", label: "八角花", category: "leaf" }
  ],
  edges: [
    { source: "embroidery", target: "pattern" },
    { source: "pattern", target: "octagon" }
  ]
}
```

## 五、验收标准

### 5.1 功能验收
- [ ] 八角花纹样 SVG 可正常显示和旋转

- [ ] 分享卡片可生成并下载

- [ ] 分享卡片生成 < 2s
- [ ] 收藏夹加载 < 500ms（100条数据）
- [ ] 首屏加载时间无明显增加
