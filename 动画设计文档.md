# 瑶绣·智问 动画与交互设计文档

## 设计理念

瑶绣是**针线艺术**，动画应体现：
- **针线穿梭感**：线条流动、轨迹动画
- **民族精致感**：细腻、优雅、不浮夸
- **文化仪式感**重要操作要有"绣成"的成就感

---

## 一、核心动画系统

### 1. 针线光标（Typing Cursor）

**设计**：光标变成一根绣花针，打字时有针线穿梭效果

```
效果描述：
- 光标样式：细长的针形，金色
- 打字时：针从上往下"刺"入，带轻微抖动
- 每输入一个字符：针线轨迹闪烁一下
- 停顿时光标轻微"呼吸"颤动，像等待刺绣
```

**技术**：CSS animation + 伪元素

---

### 2. 消息发送 - 飞针传书

**触发**：用户点击发送

**动画序列**：
1. **蓄力**（0-100ms）：发送按钮缩小到 0.9，像拉紧线
2. **发射**（100-300ms）：按钮变针形，向右上方飞入聊天区
3. **拖尾**：金色粒子拖尾，像绣线划过
4. **落定**：消息气泡从针落点"绣"出来，边缘有针线勾勒动画

**技术**：GSAP / CSS keyframes

---

### 3. AI 回复 - 绣娘织锦

**触发**：AI 开始回复

**动画序列**：
1. **起针**：瑶绣专家头像亮起金光，像拿起针线
2. **穿梭**：回复内容逐字出现，每个字有轻微"刺入"效果
3. **织锦**：重要段落（如纹样描述）伴随丝线纹理背景浮现
4. **收针**：回复完成时，八角花纹样在消息角落闪现一下

**特殊效果 - 深度思考模式**：
- 思考过程显示为**金线缠绕**效果
- 思考文字周围有轻微飘动的金丝

---

### 4. 八角花粒子系统

**场景1：页面背景**
- 轻微飘动的八角花轮廓
- 透明度 0.03-0.05，不干扰阅读
- 缓慢旋转 + 上下浮动

**场景2：重要操作反馈**
- 收藏成功：八角花从点击处"绣"出来，然后飘散
- 分享成功：八角花围绕分享卡片旋转一周

**场景3：加载状态**
- 等待回复时，中央显示旋转的八角花
- 花瓣逐片"绣"上，进度可视化

---

### 5. 收藏 - 穿针引线

**触发**：点击收藏按钮

**动画序列**：
1. **穿针**：收藏按钮变针形，向上飞出
2. **引线**：金色丝线连接到顶部收藏夹图标
3. **收线**：丝线收紧，顶部徽章数字 +1
4. **绣成**：收藏夹图标闪烁金光，像绣完一朵花

---

### 6. 主题切换 - 染色工艺

**触发**：切换主题颜色

**动画序列**：
1. **浸染**：新主题色从点击处像墨水一样扩散
2. **渗透**：颜色以波浪形式充满整个页面
3. **固色**：八角花纹样快闪，像染色完成

**技术**：CSS clip-path + CSS variables transition

---

### 7. 知识图谱 - 金丝绣图

**触发**：打开知识图谱

**动画序列**：
1. **起稿**：节点像针点一样逐点出现
2. **连线**：金丝以绘制动画连接节点
3. **饱满**：节点亮起，像绣线填满

**交互**：
- 鼠标悬停节点：金丝向四周"颤动"
- 点击节点：以该节点为中心，波纹扩散

---

### 8. 输入框 - 织机待绣

**触发**：输入框获得焦点

**效果**：
- 边框有"金线编织"动画，像准备刺绣
- 光标变成针形
- 输入时有轻微"咔嗒"感（触觉反馈）

---

## 二、粒子效果系统

### 粒子类型

#### 1. 金粉粒子（Gold Dust）

**场景**：
- 收藏成功时从按钮飘出
- 分享卡片生成时环绕
- 主题切换时飘落

**参数**：
- 大小：2-4px
- 颜色：#d4a574
- 运动：缓慢上浮 + 轻微左右摆动
- 生命周期：1-2秒

#### 2. 丝线粒子（Silk Thread）

**场景**：
- 消息发送时的拖尾
- 连线动画

**参数**：
- 长度：20-50px
- 颜色：渐变金红
- 运动：跟随主体，逐渐消失

#### 3. 八角花碎片（Octagon Fragments）

**场景**：
- 大操作成功时（如完成一系列对话）
- 页面加载完成

**参数**：
- 形状：八角花轮廓
- 大小：16-32px
- 运动：从中心向外飘散，旋转
- 透明度：逐渐减至 0

---

## 三、微交互细节

### 1. 按钮悬停

**普通按钮**：
- 放大 1.05
- 背景色渐变位移

**重要按钮（发送、收藏）**：
- 放大 1.05
- 周围出现轻微"针脚"边框动画
- 像绣了一圈金边

### 2. 滚动效果

**聊天区滚动**：
- 新消息从下方"滑入"，带轻微弹性
- 滚动到底部有"线团"动画提示

**收藏夹滚动**：
- 卡片依次淡入，像展开绣品

### 3. 数字变化

**消息计数、收藏计数**：
- 数字变化时，旧数字缩小消失，新数字放大出现
- 像翻转的绣牌

---

## 四、性能优化

### 原则
1. **60fps 优先**：复杂动画用 GPU 加速
2. **渐进增强**：低端设备减少粒子数量
3. **用户控制**：设置中可关闭动画

### 技术方案
- CSS transforms + opacity（GPU 加速）
- requestAnimationFrame 控制粒子
- 粒子池复用，避免频繁创建销毁
- prefers-reduced-motion 媒体查询支持

## 六、代码规范

### CSS 变量
```css
--anim-duration-fast: 150ms;
--anim-duration-normal: 300ms;
--anim-duration-slow: 500ms;
--anim-easing: cubic-bezier(0.4, 0, 0.2, 1);
--anim-elastic: cubic-bezier(0.68, -0.55, 0.265, 1.55);
```

### 类名规范
```
.anim-needle      - 针线相关动画
.anim-octagon     - 八角花相关动画
.anim-silk        - 丝线相关动画
.anim-pop         - 弹出动画
.anim-float       - 飘浮动画
.particle-gold    - 金粉粒子
.particle-thread  - 丝线粒子
```
